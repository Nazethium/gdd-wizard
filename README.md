# GDD Wizard:

A lightweight tool for creating professional Game Design Documents (GDDs)


GDD Wizard runs right in your browser — no installs, accounts, or downloads required
It’s fully open-source, privacy-friendly, and works both online and offline

# How to Run

Run it instantly - https://nazethium.github.io/gdd-wizard/

Or optionally, you can install it as a Progressive Web App (PWA) for quick offline access

## In Chromium based browsers

- Open https://nazethium.github.io/gdd-wizard/
- Click the install icon (a monitor with a down-arrow) on the right side of the address bar

GDD Wizard will appear in your apps list and work even without an internet connection


# Features:

GDD Wizard helps game developers quickly outline and document their projects including

## Core features
Core loop design,
Mechanics & features,
Story, setting, and characters,
Monetization & marketing plans,
Team and production notes,
Export to clean, ready-to-share PDF

## Markdown Formatting
Markdown was added to enable the use of newlines, line breaks, bold, italic, and additional headers within text fields.
This is primarily for users to better organize lists, sublists, and emphatic items while maintaining the original formatting layout of the resulting PDF.

All data stays local in your browser — nothing is uploaded or stored online


Tech Notes:

Works on desktop in browser or in a one-click installable PWA

Updates automatically when a new version is released
